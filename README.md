# Trouve ton artisan

Projet réalisé dans le cadre du devoir CEF « Créez le site Web Trouve ton artisan avec React.js ».

## Stack technique

- Frontend : React.js, Bootstrap, Sass
- Backend : Node.js, Express, Sequelize
- Base de données : MySQL
- Versionning : Git / GitHub

## Structure

- `client/` : application React
- `server/` : API Express
- `sql/` : scripts SQL
- `docs/` : maquettes, captures, dossier final
